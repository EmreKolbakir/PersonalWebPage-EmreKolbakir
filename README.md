# DynamicSocialSpace 🌌

A customizable dynamic web space template with a starry night theme. Showcase your personal social media profiles and contact information with a modern and interactive touch.

## Features 🌠
- Dynamic starry night background animation.
- Links to social media platforms.
- Dedicated contact section.

## Description 📜
This personal webpage template features a canvas-based starry background, making use of both static and moving stars to create a serene nighttime atmosphere. It also contains sections to list out social media links and a contact section. The design is responsive, ensuring it looks good on both desktop and mobile devices.

## Usage 🚀

## How to Use ⚙️

1. **Clone the Repository**
   \```bash
   git clone https://github.com/[YourUsername]/PersonalWebPage-EmreKolbakir.git
   \```
2. **Navigate to the Repository Directory**
   \```bash
   cd PersonalWebPage-EmreKolbakir
   \```
3. **Modify the Placeholders**  
   Open `index.html` and replace placeholders with your personal details. Ensure to fill in the following:
   - Social media links.
   - Contact information.
   
4. **Host Your Website**  
   Consider platforms like GitHub Pages, Netlify, or Vercel for hosting your site for free.

## Customization ✨
- You can easily change the color scheme and fonts by modifying the CSS file.
- The star count and behavior can be adjusted in the `script.js` file.
